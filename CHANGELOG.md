# Changelog

All notable changes to PA2Z are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0] - 2025-01-15

### Added
- Initial public release
- All-in-one command-line utility combining XCopy, icon conversion, delay, MIDI/audio playback, and beep tools
- Recent-file copy by date range (past weeks/days/hours/minutes, start/end date)
- SVG to ICO icon conversion with configurable sizes
- Flexible sleep timer (hours, minutes, seconds, milliseconds)
- MIDI and audio file playback with volume control
- Configurable system beeps (frequency, duration, count, pause)
